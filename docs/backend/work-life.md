# 职业规划

## 核心原则

Work-Life Balance 优先。

在保证健康、自由和个人生活质量的前提下，持续提升技术能力，获得长期稳定且具有竞争力的收入。

---

## 职业目标

成为一名具备国际化竞争力的 Senior Backend Engineer，并进入欧洲 Remote-First 或 Remote-Friendly 团队长期工作。

---

## 核心能力建设

### Engineering

* Linux
* Python
* PostgreSQL
* Redis
* RabbitMQ
* Backend Engineering

目标：

* 独立设计、开发和维护中大型后端系统
* 具备 Senior Backend Engineer 水平

---

### Architecture

* System Design
* Distributed Systems
* Software Engineering

目标：

* 具备系统设计能力
* 理解分布式系统核心原理
* 能够完成技术方案设计与架构演进

---

### English

* 阅读英文技术文档
* 编写英文技术文档
* 英文会议沟通
* 异步协作沟通

目标：

能够在纯英文环境下独立完成后端系统设计、开发与维护。

---

## 欧洲远程岗位能力

### Cloud & Deployment

* Docker
* Kubernetes
* AWS
* CI/CD

重点：

* 容器化部署
* 自动化交付
* Cloud 基础设施理解
* 日常运维与故障排查

目标：

达到能够独立部署和维护业务系统的水平。

---

## 补充能力

* React
* TypeScript

目标：

具备基础前端能力，能够独立完成个人项目和简单业务开发。

---

## 学习资源投入比例

```text
35% English

35% Python Backend

20% System Design & Distributed Systems

10% AWS / Docker / Kubernetes
```

说明：

对于欧洲远程岗位：

* Python 决定技术能力
* English 决定国际化工作能力
* System Design 决定 Senior 水平
* Cloud 能力满足岗位要求即可

---

## 理想工作状态

* 每天 6~8 小时高质量工作
* 较少加班
* 较少 On-call
* 较少非必要会议
* 拥有自主安排时间的能力
* 保持持续学习与成长

---

## 长期愿景

成为一名具备国际化竞争力的 Senior Backend Engineer，在欧洲远程团队中长期工作。

通过扎实的工程能力、英语能力和系统设计能力，获得稳定、可持续且具备良好 Work-Life Balance 的职业发展。

---

## 一句话总结

WLB 优先，深耕 Python 后端、系统设计和英语能力，以 Senior Backend Engineer 为职业发展主线，最终进入欧洲远程团队，实现长期、稳定、可持续的职业发展。

# 记录一下试用命令

## codex resume

```sh
这会打开会话选择器，让你选这个目录下的历史对话。

常用方式还有这几个：

codex resume --last

恢复这个目录下最近一次会话。

codex resume --all

显示所有目录的历史会话，不只看当前目录。

codex resume <SESSION_ID>

如果你已经知道会话 ID，直接恢复指定会话。
```
终于不用怕任务被突然打断了.
